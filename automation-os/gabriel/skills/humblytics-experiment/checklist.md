# Experiment Pre-Launch Checklist

- [ ] Hypothesis written: "If X then Y because Z"
- [ ] Control version documented (exact copy)
- [ ] Variant version documented (exact copy)
- [ ] Only ONE variable changed between control and variant
- [ ] Success metric defined
- [ ] Decision rule set (minimum sample or minimum days)
- [ ] No existing test already running on this page
- [ ] Not a repeat of a previously run test (checked logs/experiments.md)
- [ ] Logged in logs/experiments.md
- [ ] Alfred approval before launch
