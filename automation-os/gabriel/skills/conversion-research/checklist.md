# Conversion Research — Checklist

- [ ] Specific conversion question defined (not "improve conversions")
- [ ] Source is CXL, Baymard, VWO, or equivalent (not generic blog)
- [ ] Test result is specific with named data
- [ ] The WHY behind the result is extracted
- [ ] Applied to Alfred's specific page type and ICP
- [ ] Mobile applicability considered
- [ ] B2C vs. B2B context checked
- [ ] Evidence level scored (1–5)
- [ ] Risk level assessed (direct replace vs. A/B test)
- [ ] Logged in logs/conversion-insights-log.md
