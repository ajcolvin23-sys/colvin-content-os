# QA Pre-Delivery Checklist

- [ ] Compliance: no guarantees, no legal advice, housing/grant/youth flagged
- [ ] Brand: no banned openers, correct tone for platform
- [ ] Quality: no hallucinated facts, offer matches OFFER_LIBRARY, no placeholders
- [ ] Safety: no credentials, no auto-action triggers, status tag present
- [ ] Format: mobile readable, correct length, correct platform format
- [ ] Verdict issued: PASS / REVISE / FAIL
- [ ] If REVISE or FAIL: specific inline notes provided
