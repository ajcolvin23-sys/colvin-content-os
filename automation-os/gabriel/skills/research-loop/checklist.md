# Research Loop (Meta) — Checklist

- [ ] Research question clearly defined
- [ ] Sub-questions mapped to individual sessions
- [ ] Correct research skill used for each sub-question
- [ ] Evidence scored per session
- [ ] Cross-session patterns identified (3+ sessions required)
- [ ] Contradictions noted (not resolved without evidence)
- [ ] Composite evidence level assigned
- [ ] Final recommendation specific and actionable
