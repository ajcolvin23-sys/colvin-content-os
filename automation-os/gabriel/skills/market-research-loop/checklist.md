# Market Research Loop — Pre-Run Checklist

## Before Starting
- [ ] Topic is specific (not "marketing in general")
- [ ] Business lane is identified
- [ ] 20-minute time box is set

## During Research
- [ ] Using sources from RESEARCH_SOURCES.md (High tier first)
- [ ] Looking for specific examples, not general summaries
- [ ] Stopping at the time limit

## After Research
- [ ] Finding is one clear sentence
- [ ] Source is cited specifically
- [ ] Evidence level scored (1–5)
- [ ] Classification labeled (fact | inference | pattern | hypothesis)

## Before Proposals
- [ ] Evidence level ≥ 3 (if not, log only — no proposals)
- [ ] Workflow extracted if evidence ≥ 3
- [ ] Output matches RESEARCH_OUTPUT_FORMAT.md

## Logging
- [ ] Entry added to logs/research-log.md
- [ ] JSON saved to research-output/latest-daily-research.json
- [ ] Proposals routed per promotion-rules.md
