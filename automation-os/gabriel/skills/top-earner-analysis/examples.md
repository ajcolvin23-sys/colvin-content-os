# Top Earner Analysis — Examples

## Example 1: Good Run — AI Agency Operator

**Operator type:** Solo AI consulting operator (~$800K ARR)
**Lens:** Offer Architecture
**Source:** Public pricing page + documented Loom walkthrough on their site (Medium tier — named, verifiable)

**Finding:** Operator runs a "three-tier productized service" — a $500 AI audit, a $3,000 implementation package, and a $1,500/month retainer. The audit is designed to sell itself into the implementation. 80% of audit buyers convert to implementation.

**Evidence Level:** 3 — verifiable from public pricing + the operator's own documented description
**Classification:** Pattern (consistent with other productized service operators observed)

**Scale Gap:** Alfred currently has no defined audit product. Audit price point ($500) is accessible for Alfred's current client stage.

**Alfred's Application:**
Create a "30-Minute AI Workflow Audit" — $497. Audit identifies 3 manual processes to automate. Naturally positions the implementation package.

**Do This Week:** Draft the AI Workflow Audit scope and price point. Add to Colvin Enterprises offer library.
**Build Over Time:** Create the audit delivery template and report format.

## Example 2: Bad Run — Reputation Without Evidence

**What happened:** Gabriel researched a well-known LinkedIn influencer's "system" and produced a summary of their profile stats and their claims about their approach.

**Why it failed:** All evidence was the operator's own claims. No third-party verification. No documented case study. No named clients. This is Level 1 — log only.

**Rule added:** Check: has anyone ELSE documented this system working? If only the operator themselves has, it's Level 1–2.
