# CMS Pre-Delivery Checklist

Run this before handing any content draft to Alfred.

- [ ] Old version saved / documented
- [ ] Brand voice matches BRAND_GUIDE.md (direct, warm, no banned openers)
- [ ] CTA is specific and matches OFFER_LIBRARY.md
- [ ] No guarantees or inflated claims
- [ ] Compliance check complete (grant/housing/youth content flagged if applicable)
- [ ] Risk level assigned (low / medium / high)
- [ ] Recommended action stated (publish / A/B test / hold)
- [ ] Status tag: `pending_review`
- [ ] Passed to qa-publish-guard
