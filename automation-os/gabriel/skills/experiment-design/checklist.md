# Experiment Design — Checklist

- [ ] Evidence level ≥ 2
- [ ] Hypothesis in "If → Then → Because" format
- [ ] Control is exact current version
- [ ] Variant changes ONE variable only
- [ ] Primary metric is specific (not "engagement")
- [ ] Success threshold set (percentage or absolute value)
- [ ] Minimum sample size defined
- [ ] Minimum days defined (7+ minimum)
- [ ] Risk level assessed
- [ ] Compliance check for applicable lanes
- [ ] Alfred approval required before launch
- [ ] Added to logs/experiment-results.md as proposed
