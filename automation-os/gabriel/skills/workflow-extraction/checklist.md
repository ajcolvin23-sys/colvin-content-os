# Workflow Extraction — Checklist

- [ ] Evidence level confirmed ≥ 3 before starting
- [ ] Workflow name is specific and kebab-case
- [ ] Trigger condition is precise
- [ ] Each step is ONE executable action
- [ ] Decision rules cover main edge cases
- [ ] Maturity status assigned as Draft
- [ ] Saved to correct location (skill file vs. workflow file)
- [ ] Logged in logs/workflow-extraction-log.md
- [ ] Routed to Alfred for review if modifying an existing skill
