# Marketing Intelligence — Checklist

## Before Research
- [ ] Specific marketing question defined (not "marketing in general")
- [ ] Target lane identified
- [ ] Source tier selected (High or Primary only)

## During Research
- [ ] Looking for specific tactics, not summaries
- [ ] Time-boxed to 20 minutes

## Evidence Review
- [ ] Evidence level scored (1–5)
- [ ] Source tier confirmed (not social media claims)
- [ ] Classification labeled (fact | inference | pattern | hypothesis)

## Alfred's Context Check
- [ ] Brand voice fit checked against BRAND_GUIDE.md
- [ ] Compliance implications reviewed for lane
- [ ] Scale appropriateness checked (applicable to Alfred's current size)
- [ ] Channel match confirmed (ICP spends time on this channel)

## Output
- [ ] Specific application written (not generic advice)
- [ ] Channel-specific format considerations included
- [ ] Routed correctly per evidence level
- [ ] Logged in logs/marketing-insights-log.md
