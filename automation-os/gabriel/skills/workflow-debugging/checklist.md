# Workflow Debugging — Checklist

- [ ] Failure clearly stated (expected vs. actual)
- [ ] Step where failure occurred identified
- [ ] Root cause classified (bad routing | missing context | bad rule | weak checklist | tool failure | evidence skip | context bloat | compliance miss)
- [ ] Specific fix written (not vague)
- [ ] Fix routed to correct tool
- [ ] Logged in logs/failed-runs.md
