# Analytics Review Pre-Delivery Checklist

- [ ] Time period defined
- [ ] Data sources confirmed (no fabricated numbers)
- [ ] Top performers identified with specific examples
- [ ] Weak performers identified with specific examples
- [ ] Pattern stated clearly (not vague)
- [ ] Minimum 3 specific lessons extracted
- [ ] Carry-forward items reviewed and escalated if stale
- [ ] Skill update recommendations made
- [ ] Top 3 actions are specific and actionable
- [ ] No conclusions drawn from fewer than 7 data points
