# Evidence Review — Checklist

- [ ] Source specifically cited (not "I read somewhere")
- [ ] Source quality tier assigned
- [ ] Finding classified (fact | inference | pattern | hypothesis)
- [ ] Evidence level assigned (1–5)
- [ ] Disqualifiers checked (unverified stats, anonymous case, LLM assertion)
- [ ] Verdict issued
- [ ] Routed per promotion-rules.md
