# Evidence Review — Examples

## Example 1: Level 3 Verdict

**Finding:** LinkedIn connection requests under 100 words with a specific pain opener get 2.3× more replies
**Source:** CXL article, named study, 3 documented tests
**Source tier:** High
**Classification:** Pattern (across 3 tests, not a controlled experiment)
**Evidence level:** 3
**Verdict:** Propose to memory (proposed-research-memory.md) + use cautiously

## Example 2: Level 1 Downgrade

**Finding:** "Video content gets 5× more engagement than text posts"
**Source:** A LinkedIn post from an unverified account
**Source tier:** Weak
**Classification:** Hypothesis (claim without data)
**Evidence level:** 1
**Verdict:** Log only. No proposals.

## Example 3: LLM Assertion Disqualification

**Finding:** Gabriel researched a topic and produced a summary that included "Studies show that..."
**Issue:** No study was cited. This was a hallucinated citation.
**Verdict:** Disqualify. Remove the claim. Log that this type of hallucination occurred. Add anti-hallucination reminder to research checklist.
