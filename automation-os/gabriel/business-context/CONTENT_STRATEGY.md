# Content Strategy — Alfred Colvin

## The Framework on Every Piece of Content

Every post, caption, email, or page follows this structure — no exceptions:

1. **Hook** — stop the scroll. One sentence. A pattern interrupt, a contrarian truth, or a cost-of-inaction statement. No questions that start with "Are you...". No emojis in the first line.
2. **Story** — build belief. A before/after, a reframe, or a specific example. This is where conviction transfers. Do NOT fabricate client stories. Use [hypothesis] or [example scenario] labels if needed.
3. **Offer** — one clear next step. Always match the rung of the ladder being promoted. Never present the whole ladder at once.

## Diagnostic Framework (for analytics-learning-loop)

When performance is below prediction, diagnose by funnel stage:

| Symptom | Diagnosis | Fix |
|---|---|---|
| Traffic but no opt-ins/replies | Hook or offer problem | Test a new hook, clarify the offer |
| Opt-ins/calls but no sales | Story/belief problem | Strengthen the before/after, add proof |
| Sales but no ascension | Ladder/backend problem | Make the next rung obvious at point of sale |
| Low close rate | Underpriced or over-complicated offer | Reframe on value created, simplify |

## The Loop (applies to every lane, every rung)

1. Pick the one rung being worked on
2. Write Hook-Story-Offer for it
3. **Predict the metric + number before launch**
4. Launch / send
5. Measure against prediction
6. Diagnose using the table above
7. Change ONE thing, predict again, re-test
8. Only after converting: ascend to next rung or clone the funnel
