# Skill: Video Scripting

## Video Types Alfred Produces

| Format | Platform | Length | Purpose |
|---|---|---|---|
| Daily Remotion video | TikTok / Reels / Shorts | 60s vertical | First Keys Indy daily tip |
| YouTube tutorial | YouTube | 5–12 min | Gospel piano / music theory |
| Short-form reel | Instagram / TikTok | 15–60s | Business tips, AI demos |
| Long-form webinar | YouTube / LinkedIn | 20–45 min | Client education |

## Daily Remotion Video Script (First Keys Indy)

Generated by `/api/daily-video` → Remotion renders → Telegram delivery

**Script format:**
```
HOOK (0–3s): [Question or bold statement about Marion County homebuyers]
TIP (3–45s): [One specific, actionable insight about DPA programs]
CTA (45–60s): [Visit first-keys-indy.vercel.app or "Like for more"]
```

**Compliance:** Never promise specific grant amounts. Never guarantee qualification. Always say "may qualify."

## YouTube Gospel Piano Script

```
HOOK (0–30s): [Play something impressive, then ask "Want to learn this?"]
INTRO (30s–2min): [Brief bio + what today's lesson covers]
CONTENT (2–10min): [Teach one specific concept with the piano]
CTA (final 30s): ["Subscribe + comment your question" + book link or course link]
```

## Short-Form Reel Script (Business / AI)

```
0s–3s: HOOK — "Here's what most Indianapolis businesses are missing about AI"
3s–45s: CONTENT — One specific insight, show don't just tell
45s–60s: CTA — "Follow @AlfredColvin for more" + one clear next step
```

## Script Quality Rules

1. Hook must stop the scroll in 3 seconds
2. One idea per video — don't try to teach everything
3. Always show what you're talking about (demo > explain)
4. End with one clear CTA — not three options
5. No jargon without explanation
6. Compliance applies to all First Keys Indy + FundingReady content

## Model Routing

- Short scripts (< 2 min) → GPT-4o-mini (fast)
- Long scripts (5+ min YouTube) → GPT-4o (better pacing + structure)
- Always QA before adding to content queue
